## Installation

### From GitHub Releases
1.  Go to the [Releases](https://github.com/vinarius/vin/releases) page.
2.  Download the archive for your operating system and architecture:
    *   **macOS (Apple Silicon):** `*_Darwin_arm64.tar.gz`
    *   **macOS (Intel):** `*_Darwin_x86_64.tar.gz`
    *   **Linux:** `*_Linux_x86_64.tar.gz`
    *   **Windows:** `*_Windows_x86_64.zip`
3.  Extract the archive.
4.  Move the `vin` binary to a directory in your `PATH` (e.g., `/usr/local/bin` on macOS/Linux).

### Using Go
If you have Go installed:

```bash
go install github.com/vinarius/vin/v2@latest
```

## Usage
Run `vin --help` to see all available commands.
